# XianYuPlus 更新日志

本文件记录各正式版本的主要变化。项目介绍与安装说明请查看 `README.md`。

## V1.8.5（2026-07-21）

### 自动化与交付

- AI 客服回复延迟改为系统级可配置项，可在“系统设置 → AI 客服配置”设置 1–60 秒，默认 15 秒，保存后即时生效。
- 商品卡密自动发货新增“自动确认发货”开关；卡密完整发送后才会执行确认发货。
- 自动评价在待评价列表暂未返回订单时，改由闲鱼最终评价接口核验，减少需要在执行中心手动“检查并评价”的情况。

### 备份与恢复

- 扩展手动备份：账号 Cookie 与自动化设置、全部系统设置、关键词规则、通知渠道、擦亮计划、黑名单、买家标签和商品素材库均可导出与恢复。
- 备份页面增加敏感信息提示，并明确导入采用按标识新增或更新的方式，不会删除备份中不存在的旧记录。

## V1.8.1（2026-07-20）

### 界面与体验

- 运营总览的近 7 日与近 30 日交付数据改为平滑趋势图，更直观地查看每日变化。
- 将“发布商品”和“商品素材库”整理为同一组下拉导航，并调整运营中心菜单顺序。
- 系统公告移动到顶部左侧，减少与账号操作区域的视觉冲突。
- 删除订单列表、订单详情和自动化执行中心内影响布局的行内复制按钮；“更多操作”中的复制订单 ID 功能继续保留。

### 安装与更新

- Linux / 飞牛 OS 安装说明简化为一条可直接执行的安装命令。
- 移除需要额外 Docker 容器的网页在线更新功能，继续使用 `update.sh` 完成安全更新。
- 更新脚本会兼容既有数据卷，并清理此前在线更新功能可能遗留的容器和镜像。

### 文档与项目展示

- 重构项目介绍、功能说明与部署指引，让首次接触项目的用户更容易了解用途。
- 增加隐私处理后的主要功能界面预览。
- 保留交流与反馈入口，并继续强调非商业使用边界与平台风控提示。

### 质量验证

- 前端 TypeScript 类型检查与生产构建通过。
- 后端 61 项自动化测试全部通过。
